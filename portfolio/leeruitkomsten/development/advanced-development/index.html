Ik heb de specialisatie Advanced Development gevolgd waar ik meer te leren ben gekomen over het werken met JavaScript libraries door research te doen over verschillende libraries / frameworks, en daarna een project gemaakt met anime.js en jQuery.

Research

Het eerste doel van de specialisatie was meer info over libraries op te doen. Hiervoor heb ik onderzoek gedaan naar 5 verschillende libraries; Anime.js, Chart.js, Pixi.js, Three.js, en Vue.js. Voor elke library heb ik een kort project gemaakt om kennis te maken met de werking en mogelijkheden van die library.

Project

Als verdiepend project ben ik aan de slag gegaan met de library anime.js waarmee ik een animatie heb gemaakt voor het laden van webpagina's. Deze animatie heb ik daarna m.b.v. jQuery werkend gemaakt.


Eerst heb ik gekeken naar het type animatie wat ik wilde gaan maken, ik wilde het baseren op mijn logo maar had nog geen idee hoe ik dit wilde implementeren. Hiervoor ben ik aan de slag gegaan met de functies van anime.js en de documentatie bekeken voor de mogelijkheden. Zo heb ik 3 verschillende versies gemaakt en uiteindelijk tot een eindresultaat gekomen.


Voor een verdere toelichting heb ik deze video gemaakt.

Wat heb ik geleerd?

Bij het uitvoeren van deze specialisatie heb ik geleerd over verschillende libraries en het nut hiervan. Ook heb ik ervaring opgedaan met het werken met anime.js zoals het beïnvloeden van HTML elementen waarmee ik in de toekomst zeker andere animaties mee ga maken.

Wat kon beter?

In de eerste week van de specialisatie had ik de opdracht niet goed genoeg begrepen waardoor mijn onderzoek nog niet compleet was. Dit heb ik kunnen herstellen door de 2e week wél een goed onderzoek opgeleverd te hebben.
