Animations

Development
Individueel


Na portfolio review 1 kreeg ik de feedback dat ik de animaties die ik voor dit portfolio gebruikte via een JavaScript library, kon vervangen door mijn eigen JavaScript. Dit heb ik als POC gemaakt met simpele Vanilla JavaScript en wat eenvoudige CSS3 transities.


Github    Demo

Idee

De feedback om mijn eigen JavaScript animaties te maken i.p.v. een library te gebruiken sprak me erg aan. Hier ben ik me meteen in gaan verdiepen en kwam ik er achter dat dit mogelijk was met een simpel HTML element aan te roepen met een CSS class, op deze class CSS3 transities d.m.v. opacity en transform:translate de animatie te maken, en dan met JavaScript dit te laten gebeuren op de juiste scrollpositie.



Code & Feedback

Ik kreeg al snel een eerste versie in elkaar getoverd met wat simpele HTML/CSS en basis JavaScript. Hier heb ik toen samen met de development-docent gekeken naar mogelijkheden om de code te verbeteren en een bug op te lossen.


De code en methodes die ik had gebruikt werkte goed, echter was er een miniscule bug die ervoor zorgde dat de fade-up animatie bij het refreshen (dus alleen bij het geforceerd refreshen, niet bij het originele laden) ervoor zorgde dat content die deze fade-up animatie had niet volledig gecentreerd bleef. Hier heb ik nog een lange tijd samen met de docent naar gekeken maar helaas kwamen we er allebei niet uit hoe we het probleem op mosten lossen, ook al zijn we wel achter de oorzaak gekomen door te debuggen en troubleshooten.


Aangezien het zo'n kleine bug was hebben we het uiteindelijk laten liggen zodat ik er niet te lang op bleef zitten.

Wat heb ik geleerd?

Ik heb veel geleerd over JavaScript en het combineren van HTML / CSS / JavaScript door al deze elementen nodig te hebben. De gebruikte methodes waren ook nieuw, zoals de getBoundingClientRect() functie om de scrollpositie te bekijken. Ook heb ik geleerd hoe ik code kan optimaliseren door deze te doorlopen en na te denken hoe het makkelijker kan. Verder heb ik samen met de docent een troubleshoot-sessie gehad waardoor ik meer heb geleerd over het debuggen van code en het vastleggen van evenementen in de console log.

Wat kon beter?

De CSS kon wat beter geoptimaliseerd worden door classes te herbruiken om zo de minimale benodigde hoeveelheid code te gebruiken. Ook had ik meer onderzoek kunnen doen om andere methodes uit te proberen om de scrollpositie te bepalen.
