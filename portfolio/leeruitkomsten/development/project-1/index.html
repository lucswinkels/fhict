Rock Paper Scissors

Development
14/9/2020 — 15/9/2020
Individueel


Dit was de eerste POC die ik had gemaakt na het bekijken van een aantal basis JavaScript videos en mini-cursussen van w3schools. De POC is een steen-papier-schaar spel waarin animaties worden gebruikt. Dit project heb ik met behulp van een videotutorial gemaakt. Omdat ik wel mijn eigen draai wilde geven aan de tutorial heb ik een score reset functie toegevoegd en de stijl aangepast van het project.


Github

Structuur

Als eerst heb ik de HTML structuur aangemaakt. Zonder dit is er geen CSS en JavaScript om toe te passen. Hier had ik al ervaring mee dus was deze vrij logisch en snapte ik gelijk de keuzes die de tutorial had gemaakt. Deze HTML objecten worden d.m.v. hun class aangeroepen in de JavaScript.

Opmaak

Ik had al flink wat verstand van HTML & CSS dus de lay-out volgen was voor mij geen grote uitdaging. Echter heb ik wel meer geleerd over CSS animaties door keyframes aan te maken en deze binnen een bepaald object aan te roepen om dit te animeren. Er werd voornamelijk gebruikt van de transform functie om de animaties te realiseren.

Functies

Dit is natuurlijk het belangrijkste deel van de POC, de functionaliteiten die het programma heeft. In dit onderdeel heb ik ook het meest geleerd omdat voor mij het moeilijkste deel van dit project de JavaScript was. Omdat ik een tutorial heb gevolgd was dit vrij vanzelfsprekend, maar de tutorial was erg duidelijk en legt ook goed uit waarom je bepaalde functies gebruikt en wat deze functies doen. Ik heb hier bijvoorbeeld veel geleerd over variabelen aanmaken en deze aan te roepen om ze zo te veranderen.


Voor de volledige sourcecode heb ik boven in de pagina een link staan naar de Github repository en de videotutorial.
